Loni Wayfinding

32" TOUCH SCREEN INTERACTIVE WAYFINDING SOULTION

The Challenge:

Create a 49inch portrait way finding touch screen interface for LONI USC using existing brand colors and 3D floor plan mockups. Room assignments need to be updatable per semester.

Target User Persona:

College students and visiting facility or staff.

EXAMPLE USER: 20 year old female medical student trying to find classroom for the first time this semester.

EXAMPLE USER: 55 year old male or female visiting professor attempting to find classroom they are supposed to be giving a lecture in.

Solution:

Use touch screen overlay on 49inch LG digital display with Spinetix box to drive content to screen. Google doc serves as database for name and room assignments. Navigation paths were mapped out and animated in after effects then exported as .mp4 files. Touch points on the screen drive corresponding .mp4 file that illustrates path to room.

Unique Challenges:

Isolating the path from stairs to the room required us to grey out areas not applicable to intended destination. Orienting the user as to what floor they are looking at within the UI. Backend management needed to be simplistic yet robust enough to handle a very complex amount of data in a way that could be ingested by the Spinetix box and displayed on screen. User flow through the display required detailed mapping of all possible click scenarios.
